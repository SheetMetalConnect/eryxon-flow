Browse, install, and manage integrations for ERP systems and business tools. Access starter kits for custom integrations.

## Features

**For users:**
- Browse integrations with search/filter
- One-click install/uninstall
- View ratings, features, and setup docs
- Manage installed integrations

**For developers:**
- Starter kits for custom integrations
- REST API + webhooks for building integrations

## Usage

**Route:** `/admin/integrations` (Sidebar → App Store)

1. Browse available integrations
2. Click to view details (features, setup, resources)
3. Click "Install" to add to your tenant
4. View/manage in "Installed" tab

## Categories

`erp`, `accounting`, `crm`, `inventory`, `shipping`, `analytics`, `other`

## Building Custom Integrations

1. Generate API key: Admin → API Keys
2. Configure webhooks: Admin → Webhooks
3. Use [REST API](/api/api_documentation/) for data sync
4. Reference starter kits in integration detail pages

## Permissions

- **View marketplace:** All users
- **Install/uninstall:** Admins only
- **Reviews:** One per tenant per integration
