## Hosted Demo

Try Eryxon MES online with the hosted demo at [app.eryxon.eu](https://app.eryxon.eu). This is a limited version for evaluation and educational purposes only.

### Demo Limitations

| Feature | Limitation |
|---------|------------|
| Jobs | Limited number of jobs |
| Parts | Limited parts per job |
| Storage | Limited file storage for STEP/PDF files |
| API Access | Limited API requests |
| Support | Documentation only |

The hosted demo is intended for:
- **Evaluation** – Test the system before self-hosting
- **Education** – Learn how manufacturing execution systems work
- **Development** – Test integrations before deploying to your own instance

For production use, [self-host Eryxon MES](/guides/self-hosting) on your own infrastructure.
