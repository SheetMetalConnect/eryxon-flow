# Eryxon Flow Website

Landing page for [Eryxon Flow](https://github.com/SheetMetalConnect/eryxon-flow) - an open source Manufacturing Execution System for sheet metal fabrication.

## About

This is the official website for Eryxon Flow, built by [Sheet Metal Connect e.U.](https://www.sheetmetalconnect.com/) to showcase the MES platform.

## Tech Stack

- **Framework**: [Astro](https://astro.build/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Deployment**: Cloudflare Pages

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```
