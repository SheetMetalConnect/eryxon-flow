Changelog

What shipped, when.

Every release of Eryxon Flow, in plain language. Subscribe via RSS, or read the full notes for each version on GitHub.

v0.5.1
24 May 2026
Stable

Five fixes the floor will feel.

A maintenance hotfix focused on edge cases reported by Demar and Singeling during the rollouts in April. No schema changes.

Added

Bulk reassign across cells

Move up to 50 operations between cells in one move. Preserves operation order and recalculates capacity in the same tick.

Fixed

Stuck time-entry recovery

Operators leaving a tablet on overnight no longer log 14h actuals. Time entries auto-close at end-of-shift, configured per cell.

Changed

Capacity bar visualisation

Cells over 100 % planned hours now show a separate red overflow segment instead of just swapping the fill colour. Easier to read at a glance.

Added

Webhook signature verification

HMAC-SHA256 signatures on every outbound webhook. Rotate signing keys without redeploys. See docs/webhooks/signing.md.

Fixed

STEP viewer memory leak

Three.js scenes are now disposed when the operator closes the detail modal. Long sessions stay snappy.

v0.5.0
18 April 2026
Stable

QRM dashboard, planning adapters, MCP.

The biggest release since v0.3. Brings WIP limits per cell to the planner, adapters for FrePPLe and Odoo MRP, and an MCP server so AI assistants can read shop state. Includes a schema migration — read the upgrade notes before you self-host.

Added

QRM dashboard with WIP limits

Per-cell WIP limit configuration. Capacity matrix view aggregates planned vs available hours across the next 14 days.

Added

Planning adapters: FrePPLe + Odoo MRP

Two-way sync with both upstream planners. Schedule once in your planner; the floor sees it within 90 s.

Added

MCP server

Model Context Protocol server exposing read-only access to jobs, parts, and operations. Plug Claude into your shop floor.

Changed

Operator card layout

Status stripe widened to 4 px. Due-date pill moved top-right. Time-remaining now shows in minutes when < 1 h.

Removed

Legacy CSV importer

The pre-0.4 CSV format is no longer supported. Migration guide: docs/migrations/0.5-csv.md.

v0.4.7
02 March 2026
Preview

3D STEP viewer (preview).

First public preview of the browser-side STEP viewer. Behind a feature flag — enable per-tenant in Settings → Features.

Added

In-browser STEP parsing

No software install for operators. STEP files are parsed client-side via WASM; mesh rendered with Three.js.

Added

Measurement tools

Point-to-point, face distance, angle, radius. Snaps to vertex / edge / face. Orange accent matches the Eryxon palette.
