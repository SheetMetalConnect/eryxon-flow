What's in this release 4  changes
Added
Bulk reassign operations across cells

Move up to 50 operations between cells in one action. Preserves sequence order and recalculates capacity in the same tick.

Fixed
Stuck time-entry recovery

Tablets left on overnight no longer log 14 h actuals. Time entries auto-close at end-of-shift, configured per cell in settings/cells.

Changed
Capacity bar over-100 % visualisation

Cells over planned hours now show a separate red overflow segment instead of swapping the bar colour. Reads at a glance.

Added
Webhook signature verification

HMAC-SHA256 signatures on every outbound webhook. Rotate signing keys without redeploys.
